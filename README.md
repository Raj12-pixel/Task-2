# Task-2
This project focuses on data cleaning and exploratory data analysis (EDA) of a retail sales dataset to ensure data quality and generate meaningful insights. The dataset was explored to understand its structure, and issues such as missing values, duplicate records, and inconsistent formats were identified and handled.

Key tasks included handling missing data, removing duplicates, standardizing data formats, and detecting outliers to maintain data integrity. Descriptive statistics such as mean, median, mode, and standard deviation were calculated to summarize the dataset.

Visualizations like bar charts and line charts were created to analyze patterns and trends in the data. The project demonstrates how proper data cleaning and analysis can improve data reliability and support better decision-making.
